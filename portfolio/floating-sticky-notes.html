Project Overview

Floating Sticky Notes provides a flexible and convenient way to capture thoughts, reminders, and important information with colorful, customizable notes that float above other applications for easy access.

Key Features

  • Floating sticky notes displayed anywhere on screen
  • Pin notes to board for persistent display
  • Unlimited note creation and management
  • Colorful and customizable note appearance
  • Rich text support for typing and creating notes
  • Minimize, edit, and share notes functionality
  • Show notes anywhere on the device for quick access

Tech Stack

Swift, UIKit, Core Data, window overlay APIs
