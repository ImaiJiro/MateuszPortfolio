Project Overview

A comprehensive delivery management application that connects drivers and senders for efficient package delivery services. The platform features dual interfaces for both drivers and senders, real-time tracking, bidding system, and integrated mapping functionality.

Key Features

  • Dual user interface for drivers and senders
  • Real-time delivery tracking and management
  • Bidding system for delivery requests
  • Integrated Google Maps for route planning and navigation
  • Delivery history and status tracking (Pending, Current, Past)
  • Earnings tracking and payout management system
  • Push notifications for delivery updates
  • Driver and sender profile management
  • Distance and time estimation for deliveries
  • Rating and review system for drivers
  • Payment processing with bank transfer integration

Tech Stack

Swift, UIKit, MapKit, CoreLocation, REST APIs, push notifications, Core Data
