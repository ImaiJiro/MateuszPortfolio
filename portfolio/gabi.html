Technical Overview

Gabi is a native iOS application developed for a Las Vegas pastry bakery. The app focuses on driving repeat visits and higher average order value through a customer loyalty and rewards system, streamlined mobile ordering, and a clear, intuitive storefront experience.

Core Features

  • Loyalty & Rewards Program – Points-based system with tiered rewards, redeemable for discounts and free items
  • Mobile Ordering – Browse menu, customize items, and place orders for pickup or delivery
  • Repeat Purchase Incentives – Promotions, stamps, and offers designed to increase visit frequency
  • Order History & Tracking – Past orders and status updates for a smooth pickup flow

Architecture & Implementation

Built with Swift and UIKit using an MVVM architecture. The app integrates with backend services for real-time menu data, loyalty balances, and order management. Secure authentication and payment-ready flows support a production-ready retail experience.

Tech Stack

Swift, UIKit, Core Data, REST APIs, Firebase (optional), modular architecture, secure authentication
