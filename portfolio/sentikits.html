Technical Overview

SentiKits is a cross-platform mobile app (iOS & Android) built with React Native. It combines structured reflection tools, AI-assisted learning, and guided emotional exercises. The app targets users navigating grief, breakups, anxiety, or major life transitions—delivering scalable, personalized support through a self-guided mobile experience.

Architecture & Implementation

Developed with React Native, React, and JavaScript. Backend powered by Express/Node.js. Core features include:

  • AI-enabled reflection modules – Personalized prompts and adaptive learning flows driven by user input
  • Intelligent journaling – Structured self-reflection with context-aware suggestions
  • Digital learning frameworks – Modular content delivery for emotional skills and coping strategies
  • Backend integration – REST APIs for content sync, user progress, and AI model coordination

My Contribution

Led design and implementation of AI-enabled learning and reflection features. Delivered structured reflection modules, intelligent journaling concepts, and digital learning frameworks that improve the platform’s ability to provide personalized emotional wellbeing tools at scale.
